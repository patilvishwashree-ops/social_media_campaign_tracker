# Social Media Campaign Performance Tracker

## Project Overview
This project analyzes Facebook Ads campaign data to evaluate performance, engagement, CTR, CPC, and ROI using Python.

## Tools & Technologies
- Python
- Pandas
- Matplotlib
- Plotly
- Streamlit

## KPIs Analyzed
- Impressions
- Clicks
- CTR
- CPC
- Spend
- Engagement
- Conversions
- ROI (assumed revenue model)

## Key Insights
- Ad spend does not directly guarantee higher performance.
- Lower CPC campaigns tend to have better CTR.
- Engagement is higher than actual conversions.
- ROI indicates scope for campaign optimization.

## Dashboard
An interactive Streamlit dashboard is created to visualize trends and performance metrics.

## How to Run
1. Install dependencies  
   `pip install -r requirements.txt`
2. Run analysis  
   `python analysis.py`
3. Launch dashboard  
   `streamlit run dashboard.py`
